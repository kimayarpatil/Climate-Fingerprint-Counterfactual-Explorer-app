# Climate-Fingerprint-Counterfactual-Explorer-app
# 🌍 Climate AI Pro

An AI-powered multi-page Streamlit application for environmental impact analysis using object detection and data analytics.

## 🚀 Features
- 📊 EDA Dashboard
- 🖼️ Image Object Detection (YOLOv8)
- 🎥 Live Webcam Detection
- 🔁 Counterfactual Simulation
- 📈 Emission Predictor
- 📄 PDF Report Generation

## 🛠️ Tech Stack
- Python
- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV
- Pandas, Matplotlib

## ▶️ Run Locally

bash
pip install -r requirements.txt
streamlit run app.py
